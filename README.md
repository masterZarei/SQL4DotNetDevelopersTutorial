# 🎓 دوره جامع آموزش SQL Server برای توسعه‌دهندگان ASP.NET Core

سلام! خوش آمدید به ریپازیتوری رسمی دوره جامع آموزش SQL Server ویژه‌ی برنامه‌نویسان ASP.NET Core.  
این دوره به شکلی کاملاً **کاربردی، پروژه‌محور و از صفر تا پیشرفته** طراحی شده است تا توسعه‌دهندگان ASP.NET Core را به مهارت کامل در SQL Server مجهز کند.
